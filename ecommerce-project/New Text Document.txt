 نظام المستخدمين والمصادقة
- تسجيل المستخدمين مع التحقق من صحة البيانات
- تسجيل الدخول والخروج
- دور خاص للمشرف مع صلاحيات إضافية
 

 إدارة المنتجات
- عرض المنتجات بطريقة جذابة
- البحث والتصفية حسب الاسم، الفئة، أو السعر
- إضافة/تعديل/حذف المنتجات (للمشرف فقط)


سلة التسوق والطلبات
- إضافة المنتجات إلى السلة
- تعديل الكميات وإزالة المنتجات من السلة
- إتمام عملية الشراء وإنشاء طلب
- عرض تاريخ الطلبات للمستخدم


لوحة تحكم المشرف
- عرض مؤشرات الأداء (عدد المنتجات، عدد الطلبات، إجمالي المبيعات، عدد المستخدمين)
- إدارة الطلبات وتحديث حالتها
- عرض تفاصيل الطلبات


التقنيات المستخدمة

- **HTML5**: لبناء هيكل الصفحات
- **CSS3**: لتنسيق وتصميم الواجهة
- **JavaScript**: للبرمجة وإضافة التفاعلية
- **Bootstrap 5**: لتصميم واجهة متجاوبة
- **Font Awesome**: لإضافة الأيقونات
- **localStorage**: لتخزين البيانات محلياً

طريقة الاستخدام


 للمستخدمين العاديين
1. قم بإنشاء حساب جديد أو تسجيل الدخول
2. تصفح المنتجات واستخدم خيارات البحث والتصفية
3. أضف المنتجات إلى سلة التسوق
4. اضبط الكميات أو أزل المنتجات من السلة حسب الحاجة
5. أتمم عملية الشراء بإدخال عنوان الشحن وطريقة الدفع
6. راجع تاريخ طلباتك وحالتها


للمشرفين
1. سجل الدخول باستخدام حساب المشرف (اسم المستخدم: admin، كلمة المرور: admin123)
2. استخدم لوحة التحكم لمراقبة مؤشرات الأداء
3. أضف، عدل، أو احذف المنتجات
4. راجع الطلبات وقم بتحديث حالتها
